# 小工宝需求文档


#### 整体描述
> 搭建建筑（我也不清楚是土建还是啥）行业的信息分享平台，解决建筑行业信息不对等问题，同时为行业人员提供交流的社区。  
> #### 展示方式
> 1.地图页面：显示使用者周围发布的各种信息可以筛选工种类型。      
 点击地图标注，跳转到发布信息的详情页面（如果同一个位置周围太多信息将会是列表页呈现）
